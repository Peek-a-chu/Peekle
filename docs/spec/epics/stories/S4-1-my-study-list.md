## 🧾 User Story

**As a** 사용자
**I want to** 내가 참여 중인 스터디 방 목록을 조회하고 검색하고 싶다

## ✅ Acceptance Criteria

1. **목록 조회**
   - 기본적으로 모든 스터디는 비공개이며, 목록에는 내가 이미 참여한 스터디만 표시되어야 한다.

2. **스터디 카드 정보**
   - 각 카드에는 다음 정보가 표시되어야 한다:
     - 방 제목, 설명
     - 총 참여 인원 수
     - 스터디 그룹 랭킹 포인트 (점수)
     - 방장 프로필 이미지 및 닉네임, 내가 방장인지 여부(뱃지 등)
     - 현재 온라인인 참여자들의 프로필 이미지 목록 (Avatar Pile)

3. **검색**
   - 상단 검색창을 통해 "내 목록 내에서" 스터디 제목으로 검색(필터링)할 수 있어야 한다.

4. **화면 구성**
   - 상단에 '참여하기(코드 입력)', '방 만들기' 버튼이 있고, 하단에 스터디 카드 그리드가 배치된다.

5. **입장**
   - 리스트의 카드를 클릭하면 즉시 해당 스터디 룸(장치 설정 화면)으로 이동한다.
