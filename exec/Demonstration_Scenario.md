# 시연 시나리오 (Demonstration Scenario)

본 문서는 프로젝트의 핵심 기능을 시연하기 위한 순서와 절차를 설명합니다.

## 1. 로그인 및 메인 화면

### 1.1. 소셜 로그인
1.  **접속**: 웹 브라우저를 열고 서비스 URL에 접속합니다.
2.  **로그인**: 메인 화면의 '로그인' 버튼을 클릭합니다.
3.  **서비스 선택**: Kakao, Naver, Google 중 하나의 소셜 로그인 버튼을 클릭합니다.
4.  **인증**: 해당 소셜 서비스의 로그인 창에서 인증을 완료합니다.
5.  **결과**: 로그인이 완료되면 메인 대시보드 화면으로 이동합니다. 우측 상단에 사용자 프로필 이미지가 표시되는지 확인합니다.

### 1.2. 메인 대시보드 확인
1.  **정보 확인**: '내 정보' 섹션에서 닉네임, 티어(아이콘), 포인트 정보를 확인합니다.
2.  **활동 로그**: 최근 문제 풀이 기록이나 활동 그래프가 정상적으로 표시되는지 확인합니다.

## 2. 게임 룸 생성 및 입장

### 2.1. 방 만들기 (호스트)
1.  **방 만들기 클릭**: 메인 화면의 '방 만들기' 버튼을 클릭합니다.
2.  **옵션 설정**:
    - **제목**: "시연용 게임방" 입력
    - **모드**: "스피드전" 선택
    - **인원**: 4명 설정
    - **비공개 여부**: 공개 (비밀번호 없음)
3.  **생성**: '생성하기' 버튼을 클릭합니다.
4.  **대기실 입장**: 게임 대기실로 이동하며, 호스트 권한(왕관 아이콘 등)이 있는지 확인합니다.

### 2.2. 친구 초대 및 입장 (게스트)
1.  **초대 링크 복사**: 대기실 내 '초대 링크 복사' 버튼을 클릭합니다.
2.  **입장**: (시크릿 탭 등을 이용하여) 다른 브라우저 창에서 복사한 링크로 접속하거나, 메인 리스트에서 해당 방을 찾아 클릭합니다.
3.  **준비**: 게스트는 '준비 완료' 버튼을 클릭하여 상태를 변경합니다.

## 3. 게임 진행 (Peekle 모드)

### 3.1. 게임 시작
1.  **시작**: 모든 플레이어가 '준비 완료' 상태가 되면, 호스트가 '게임 시작' 버튼을 누릅니다.
2.  **카운트다운**: 3, 2, 1 카운트다운 후 문제 화면으로 전환됩니다.

### 3.2. 문제 풀이 및 화상 채팅
1.  **화상 통신 확인**: 화면 하단 또는 거울 모드를 통해 카메라와 마이크가 작동하는지 확인합니다(LiveKit).
2.  **문제 확인**: 좌측의 알고리즘 문제 설명을 확인합니다.
3.  **코드 작성**: 우측 코드 에디터에 정답 코드를 작성합니다.
4.  **제출**: '코드 제출' 버튼을 클릭하여 채점을 진행합니다.
5.  **아이템 사용**: (아이템전인 경우) 상대방을 방해하거나 자신에게 유리한 아이템을 사용합니다.

### 3.3. 게임 종료
1.  **결과 확인**: 제한 시간이 끝나거나 목표를 달성하면 게임이 종료됩니다.
2.  **순위표**: 최종 순위와 점수 변동 화면을 확인합니다.
3.  **나가기**: '대기실로 돌아가기' 또는 '메인으로 나가기'를 클릭합니다.

## 4. 마이페이지 및 기록

### 4.1. 전적 확인
1.  **프로필 이동**: 메인 화면 우측 상단의 프로필 아이콘을 클릭하여 '마이페이지'로 이동합니다.
2.  **전적 조회**: 최근 게임 전적, 승률, 풀었던 문제 목록이 갱신되었는지 확인합니다.

### 4.2. 설정 변경
1.  **정보 수정**: 닉네임 변경이나 프로필 이미지 업로드를 시도하고 변경 사항이 저장되는지 확인합니다.
