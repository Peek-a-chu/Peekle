# Peekle: Baekjoon Submit Tracker Extension

**힐끔힐끔코딩 (Peekle)** 크롬 확장 프로그램입니다.
백준 온라인 저지(BOJ)에서 문제를 풀면 자동으로 감지하여 내 풀이 기록을 관리할 수 있습니다.

## ✨ 주요 기능

1.  **자동 제출 감지**
    *   백준 '채점 현황' 페이지에서 "맞았습니다!!" 결과가 나오면 자동으로 감지합니다.
    *   채점 대기/채점 중 상태를 추적하여 결과가 나오는 즉시 반응합니다.
    *   이미 지나간 과거의 제출 내역은 무시하고, 최근(방금 푼) 내역만 기록합니다.

2.  **풀이 정보 수집**
    *   문제 번호, 제목, 난이도 (Solved.ac 연동)
    *   사용 메모리(KB), 실행 시간(ms), 사용 언어
    *   **소스 코드 자동 수집**: 내가 제출한 코드를 자동으로 가져와 저장합니다.

3.  **개발용 테스트 도구**
    *   제출 시뮬레이션: `tester.html`을 통해 외부 사이트에서 백준으로 코드를 전송하고 자동 입력하는 기능을 테스트할 수 있습니다.
    *   주소: `chrome-extension://<ID>/tester.html`

## 🛠 설치 및 실행 방법

1.  이 폴더(`apps/extension`)를 준비합니다.
2.  Chrome 브라우저에서 `chrome://extensions` 접속.
3.  우측 상단 **"개발자 모드"** 켜기.
4.  **"압축해제된 확장 프로그램을 로드합니다"** 클릭 후 이 폴더 선택.

## 📂 프로젝트 구조

*   `manifest.json`: 확장 프로그램 설정 및 권한 정의
*   `background.js`: 백그라운드 서비스 워커 (알림 표시, 데이터 저장, API 호출 등)
*   `content.js`: 백준 페이지 내 스크립트 (채점 감시, 소스 코드 추출, 자동 입력)
*   `popup.html` / `popup.js`: 확장 프로그램 아이콘 클릭 시 나오는 팝업 UI
*   `tester.html` / `tester.js`: 외부 연동 기능 테스트용 페이지

## 📝 개발 노트

*   **권한**: `storage`, `notifications`, `alarms` 권한을 사용합니다.
*   **호스트 권한**: `acmicpc.net` (백준), `solved.ac` (난이도 정보)에 접근합니다.
